# SwordsSMP Combined with Armour Invis

This is a combined Minecraft Fabric mod for Minecraft 1.21.11 that includes both:
- **SwordsSMP** - Custom swords and items mod
- **Armour Invis** - Armor invisibility when player has Invisibility II effect

## Features

### From SwordsSMP:
- Custom swords with special abilities
- Custom items and tools
- Potions and brewing recipes
- Custom mob effects
- Game rules modifications

### From Armour Invis:
- Hides worn armor when a player has Invisibility II
- Client-side rendering modifications
- Requires the mod on client to hide armor visuals

## Building

To build the mod, use the Gradle wrapper:

```bash
./gradlew clean build
```

The compiled JAR will be located in `build/libs/Swordsmp & Armourinvis-1.0.0.jar`

## Running the Client

To run the Minecraft client with the mod:

```bash
./gradlew runClient
```

## Requirements

- Minecraft 1.21.11
- Fabric Loader 0.18.4 or higher
- Fabric API 0.141.2+1.21.11 or higher
- Java 21 or higher

## Installation

1. Install Fabric Loader for Minecraft 1.21.11
2. Place `Swordsmp & Armourinvis-1.0.0.jar` in your `.minecraft/mods` folder
3. Launch Minecraft with the Fabric profile
